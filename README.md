# Test Data Generation Scripts
PowerShell scripts for generating test files to simplify QA workflows

## Scripts: 
- [salesreport-generator.ps1](http://github.com/alles-liebe/test-data-generation-ps-scripts/blob/main/salesreport-generator/salesreport-generator.ps1)
- [salesreport-return-generator.ps1](https://github.com/alles-liebe/test-data-generation-ps-scripts/blob/main/salesreport-return-generator/salesreport-return-generator.ps1)

## Features
**salesreport-generator.ps1:**
- Read provided XML file
- Generate CSV files based on the provided XML file (with hardcoded items number <= 3)
- Support configurable parameters: location, shipping type and number of items

**salesreport-return-generator.ps1:**
- Read provided CSV file generated by `salesreport-generator.ps1` with TYPE = SHIP only
- Generate CSV file based on it 
- Support configurable parameters: number of items

## Requirements
- PowerShell 7+
- Execution policy enabled:<br/>
`Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass`
- Order.xml file:
 example: [order-zafn-7395638020.xml](https://github.com/alles-liebe/Test-Data-Generation-Scripts/blob/main/salesreport-generator/order-zafn-7395638020.xml)
