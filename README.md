# Test Data Generation Scripts
PowerShell scripts for generating test files to simplify QA workflows

## Scripts: 
- salesreportGenerator.ps1
- salesreportReturnGenerator.ps1

## Features
**salesreportGenerator.ps1:**
- Read XML file located on the server
- Generate CSV files based on the provided XML file (with hardcoded items number <= 3)
- Support configurable parameters: location, shipping type and number of items

**salesreportReturnGenerator.ps1:**
- Read CSV file located on the server, generated by the salesreportGenerator.ps1
- Generate CSV file based on it 
- Support configurable parameters: number of items

## Requirements
- PowerShell 7+
- Execution policy enabled:<br/>
`Set-ExecutionPolicy RemoteSigned`
- Order.xml file:
 example: [salesreport_zafn_2025-11-17T124156.xml](https://github.com/alles-liebe/Test-Data-Generation-Scripts/blob/main/Order_zafn_7395638020093069.xml)
